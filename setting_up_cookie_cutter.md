#Setting up using cookie cutter

(Setting up cookie cutter)[https://cookiecutter-django.readthedocs.io/en/latest/2-local-development/developing-locally.html]

Cookie cutter is a common template to set up a django project for
production, so we will use that.

## Set python version

    uv python pin 3.12

## Install cookiecutter

    uv pip install cookiecutter==2.6.0
